===============================================

INSTALLATION:
-------------
1. Localisez votre dossier Hytale:
   - Ouvrez le launcher Hytale
   - Cliquez sur "Settings" (en haut à gauche)
   - Cliquez sur "Open Directory"
   
2. Naviguez vers: UserData/Packs/
   (Si le dossier Packs n'existe pas, créez-le)

3. Copiez le dossier "FrenchPack" dans ce répertoire Packs
   Le chemin final devrait être:
   ...Hytale/UserData/Packs/FrenchPack/

4. Lancez Hytale

5. Activez le pack:
   - Allez dans l'onglet "Worlds"
   - Clic droit sur le monde où vous voulez la traduction
   - Activez "FrenchTranslation" dans la liste des packs
   - Entrez dans le monde
